# Corset

## Description

Authored with PBR metallic/smoothness intended for import into Unity3D.

Binary FBX contains only the geometry. Textures are in separate files.

|Name|Map|
|----|---|
|`Corset_A`|Albedo|
|`Corset_M`|Metallic/Smoothness|
|`Corset_N`|Normal|
|`Corset_O`|Ambient Occlusion|
